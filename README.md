## Nest Server
A backend server based on NestJS and MySQL

## Installation
```bash
$ npm install
```

## Configuration  
Copy the `_.env` file to `.env` and complete the configurations. Don't commit the `.env` file, Make it private!

## Running the app
```bash
# development (localhost:3000)
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
